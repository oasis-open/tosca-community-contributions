# io.terraform:1.0

TOSCA profile for [Terraform](https://www.terraform.io), HashiCorp's
infrastructure-as-code tool.

## Types

- **Terraform** (node type) — an installation of the Terraform CLI.
- **Terraform** (artifact type) — a Terraform configuration file,
  with `action` (create/modify/delete) and `tfstate` properties.
